# KQL Hunting Queries

A curated collection of custom Kusto Query Language (KQL) scripts built for cyber threat hunting, incident response, and defensive analytics across Microsoft Sentinel, Microsoft Defender, and other Azure data sources. This repository provides ready-to-use and field-tested hunting queries maintained by a Cyber Threat Hunter and Analyst, focused on strengthening detection engineering and proactive defense through practical, high-fidelity KQL.
